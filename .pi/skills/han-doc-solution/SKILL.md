# 고민 해결 방안 자동 생성

고민 해결 방안 HWPX 템플릿의 누름틀을 AI가 자동으로 채운다.

## 누름틀 목록

| 누름틀 이름 | 설명 |
|------------|------|
| `고민해결방안_제목` | 리포트 제목 (약 20자, 재치있는 표현) |
| `고민해결방안_요약` | 고민 상황 요약 (약 200자) |
| `고민해결방안_본문` | 6개 목차 구조 본문 |

## Step 1: 입력값 수집

프롬프트에 주제가 포함되어 있으면 바로 사용한다.
포함되어 있지 않으면 사용자에게 질문한다.

**필수 입력:**
- `topic`: 고민 주제 (예: "업무 집중력이 자꾸 떨어진다", "지출이 너무 많다")

**선택 입력 (없으면 생략):**
- `coreContents`: 고민 상황 설명 (예: "저녁을 먹어도 야식이 당긴다", "매달 예산을 세워봤지만 잘 안 된다")

사용자가 주제만 입력한 경우:
> "고민 상황을 더 자세히 설명해 주시겠습니까? (없으면 Enter)"

## Step 2: HWPX 문서 열기

```
mcp__hwp_file__hwp_file_open_document(filePath: ".pi/han-doc/templates/고민 해결 방안.hwpx")
```

## Step 3: 누름틀별 내용 생성 및 채우기

각 누름틀에 대해 순차적으로:
1. 누름틀의 용도에 맞는 내용을 생성
2. `mcp__hwp_file__hwp_file_fill_click_field` 도구로 누름틀에 채움

### 생성 지침

- 한국어, 친근하고 공감 가는 어투 사용
- 딱딱한 전문 용어보다 일상적인 표현 선호
- 실용적이고 실행 가능한 해결책 중심
- 불필요한 이모지, 과도한 수식어 배제

**고민해결방안_제목**: 약 20자, 마침표 없음, 재치있는 상징적 표현 허용 (말장난, 이중적 표현 가능)

```
mcp__hwp_file__hwp_file_fill_click_field(fieldName: "고민해결방안_제목", value: "{생성된 제목}")
```

**고민해결방안_요약**: 약 200자, 짧은 문장 중심, 말 걸듯 친근한 톤
- 구성: 고민 상황 요약 1문장 + 공감/문제 핵심 요약 1~2문장 + 해결 방향 요약 1~2문장
- 주제를 다시 언급하지 않음

```
mcp__hwp_file__hwp_file_fill_click_field(fieldName: "고민해결방안_요약", value: "{생성된 요약}")
```

**고민해결방안_본문**: 6개 고정 목차 구조로 작성
- 목차: **1. 고민 정의 / 2. 원인 파악 / 3. 목표 설정 / 4. 해결 전략 / 5. 유지와 성장 / 6. 결론**
- 각 목차별 2 depth: 번호 매긴 핵심 주제 문장 (예: 1-1, 1-2)
- 각 목차별 3 depth: 세부 문장 3~5개, `-` 기호로 작성

```
mcp__hwp_file__hwp_file_fill_click_field(fieldName: "고민해결방안_본문", value: "{생성된 본문}")
```

## Step 4: 문서 저장

topic을 기반으로 파일명을 생성한다 (공백→밑줄, 특수문자 제거).

```
mcp__hwp_file__hwp_file_save_document(filePath: ".pi/han-doc/output/고민해결방안_{topic_sanitized}.hwpx")
```

완료 메시지:
> "고민 해결 방안 리포트가 생성되었습니다. .pi/han-doc/output/고민해결방안_{topic_sanitized}.hwpx 파일을 확인하세요."
