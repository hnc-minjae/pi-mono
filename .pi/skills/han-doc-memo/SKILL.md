# 공문 자동 생성

공문 HWPX 템플릿의 누름틀을 AI가 자동으로 채운다.

## 누름틀 목록

| 누름틀 이름 | 설명 |
|------------|------|
| `공문_제목` | 공문 제목 |
| `공문_개요` | 공문 개요/요지 |
| `공문_상세내용` | 상세 내용 |

## Step 1: 입력값 수집

프롬프트에 주제가 포함되어 있으면 바로 사용한다.
포함되어 있지 않으면 사용자에게 질문한다.

**필수 입력:**
- `topic`: 공문 주제 (예: "사무실 이전 안내")

**선택 입력 (없으면 생략):**
- `kind`: 공문 유형 — `운영계획`, `안내공지`, `협조요청` 중 택1 (기본: 안내공지)
- `coreContents`: 핵심 내용 (예: "3월 15일부로 5층에서 7층으로 이전")
- `keyword`: 키워드 (최대 3개)

사용자가 주제만 입력한 경우:
> "공문 유형을 선택해 주세요: 1) 운영계획 2) 안내공지 3) 협조요청 (기본: 안내공지)"
> "핵심 내용을 추가로 입력하시겠습니까? (없으면 Enter)"

## Step 2: HWPX 문서 열기

```
mcp__hwp_file__hwp_file_open_document(filePath: "문서/공문.hwpx")
```

## Step 3: 누름틀별 내용 생성 및 채우기

각 누름틀에 대해 순차적으로:
1. 누름틀의 용도에 맞는 내용을 생성
2. `mcp__hwp_file__hwp_file_fill_click_field` 도구로 누름틀에 채움

### 생성 지침

- 공문 유형(`kind`)에 맞는 어투와 구성 사용
- 한국어, 공식 문서 어투, "~합니다" 체 사용
- 불필요한 이모지, 수식어 배제

**공문_제목**: "~에 관한 건", "~안내" 등 공문 제목 형식 1줄
**공문_개요**: 2~3문단, 공문의 배경과 요지
**공문_상세내용**: 3~5문단, 구체적 내용 (일시, 장소, 대상, 절차 등 포함)

각 누름틀 채우기:
```
mcp__hwp_file__hwp_file_fill_click_field(fieldName: "공문_제목", text: "{생성된 내용}")
```

## Step 4: 문서 저장

```
mcp__hwp_file__hwp_file_save_document()
```

완료 메시지:
> "공문이 생성되었습니다. 문서/공문.hwpx 파일을 확인하세요."
