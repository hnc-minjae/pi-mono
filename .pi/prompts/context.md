# 한컴 개발 환경

## 개인 설정

`~/.config/han/han-config.md` 파일에 사용자 개인 정보와 환경 설정이 있다.
세션 시작 시 이 파일을 읽어 사용자의 이름, 소속, 이메일 등을 파악한다.

## 사용 가능한 도구

### MCP Atlassian 도구 (mcp__atlassian__*)
Jira/Confluence 접근이 가능하다. 이슈 조회, 검색, 생성, 상태 전환, 코멘트, Confluence 페이지 등.

### 기본 도구
bash, read, edit, write, grep, find, ls 등 파일 시스템과 셸 명령.

## 응답 언어
항상 한국어로 응답한다.
